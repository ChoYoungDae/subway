# 동적 출구 선택 로직 (Dynamic Exit Selection) — 구버전

## 배경 및 목적
기존의 경로 탐색 프로세스는 사용자가 `SearchingView`에서 시작역과 도착역을 검색할 때 **출입구(Exit)를 먼저 선택**해야 하는 구조였습니다. 
이로 인해 다음과 같은 문제점이 발생했습니다:
1. 로컬 DB(`elevators` 등)와 교통약자 이동경로 KRIC API(`handicapped/stationMovement`) 간의 출구 데이터 불일치.
2. 상하행선 게이트가 분리된 역의 경우, **사용자가 가야 할 방향(목적지)에 따라 이용할 수 있는 출입용 엘리베이터가 다름**에도 불구하고 검색 단계에서는 방향을 알 수 없어 모든 출구를 노출함. (잘못된 출구를 선택하면 경로 탐색 실패 에러 발생)

이러한 문제를 근본적으로 해결하기 위해, 경로를 먼저 탐색하여 **탑승할 방향을 확정한 뒤 해당 방향과 연결된 출구 목록만 유동적으로 뽑아내어 선택하게 하는 구조**로 리팩토링했습니다.

## 핵심 프로세스 (개편된 플로우)

### 1단계: 역 지정 (SearchingView)
* 사용자는 검색창에서 역 이름만 검색하고 선택합니다. (출구 선택 단계 제거)

### 2단계: 최단 경로 탐색 (RouteService)
* `Find Step-Free Route` 버튼을 클릭하면 출발역과 도착역 정보를 Data.go.kr API로 보내 최단 경로(Shortest Path) 노선을 계산합니다.
* 이 단계에서 출발역에서 타야 할 노선의 방향(다음 역)과, 도착역에서 내리는 노선의 방향(이전 역)이 확정됩니다.

### 3단계: 방향 기반 유효 출구 추출 (RouteAssembler)
* 경로가 계산된 후 KRIC의 `handicapped/stationMovement` API를 호출합니다.
* **출발역(Origin)**: `stationMovement`의 전체 이동 동선 중 탑승 플랫폼 방향(다음 역)으로 이어지는 동선만 필터링한 후, 남은 동선들이 어떤 지상 출구 번호(Exit)에서 시작하는지 파싱합니다.
* **도착역(Destination)**: 하차 플랫폼(이전 역)에서 지상으로 나가는 동선만 필터링하고, 최종적으로 연결되는 출구 번호들을 파싱합니다.

### 4단계: 출구 정보 입력 지연 및 모달 제공 (RoutePreviewScreen)
* `RouteAssembler`가 추출한 유효 출구 목록을 기반으로 **모드(mode)에 따라** UI 분기 처리를 합니다:

#### Planning 모드 (`mode === 'planning'`)
* 출구 선택 모달을 **표시하지 않음** — 출구 수에 관계없이 자동으로 첫 번째 출구를 적용해 즉시 경로 결과를 보여줍니다.
* 결과 화면에서 **출발역 진입 단계(originSteps)와 도착역 진출 단계(destinationSteps)를 표시하지 않음**.  
  → 출발역 승강장 ↔ 도착역 승강장 구간(열차 탑승 + 환승)만 표시합니다.
* 결과 하단에 **Check Exit Elevators** 카드를 표시하여 도착역 출구별 엘리베이터 상태를 참고 정보로 제공합니다.

#### On-site 모드 (`mode === 'onsite'`)
* 기존 로직 유지:
  - **유효 출구가 1개뿐일 때**: 자동 적용.
  - **유효 출구가 2개 이상일 때**: 모달을 띄워 사용자에게 묻습니다.
    * _"정확한 방향 탑승을 위해 이용하실 출발역 출구를 선택해 주세요: [1번] [3번]"_
    * _"목적지로 나갈 도착역 출구를 선택해 주세요: [2번] [4번]"_
* 결과 화면에서 출발역 진입 단계 + 도착역 진출 단계를 포함한 **전체 동선**을 표시합니다.

### 5단계: 배리어프리 경로 조립 완료 (RouteAssembler)
* 사용자의 최종 출구 선택(또는 자동 선택된 출구) 값이 결정되면, 이 값을 기반으로 번역(`translate-movement`) 및 엘리베이터 가동 현황 처리를 수행하고 경로 상세 카드(Timeline)를 완성합니다.

## 모드별 표시 범위 요약 (구버전)

| 구간 | Planning | On-site |
|------|----------|---------|
| 출발역 진입 (출구 → 승강장) | ❌ 미표시 | ✅ 표시 |
| 열차 탑승 + 환승 | ✅ 표시 | ✅ 표시 |
| 도착역 진출 (승강장 → 출구) | ❌ 미표시 | ✅ 표시 |
| 출구 선택 모달 | ❌ 없음 | ✅ 복수 출구 시 표시 |
| Check Exit Elevators 카드 | ✅ 표시 | ❌ 없음 |
